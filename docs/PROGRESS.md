## Setup & Running Instructions

### Prerequisites
```bash
Node.js 18+
Python 3.11+
MongoDB 6.0+
```

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with your MongoDB URI
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### ML Module Setup
```bash
cd ml-module
pip install -r requirements.txt
python src/preprocessor.py  # Test run
python src/recommender.py   # Test run
```

### Access Application
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000
- Health Check: http://localhost:5000/api/health
