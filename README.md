# NeuroLearn: AI-Powered Adaptive Learning Platform

## Project Overview
An intelligent learning platform designed specifically for neurodiverse students, using AI/ML to adapt content delivery, pacing, and interface based on individual learning patterns.

## Team Members
- **Aakash Khandelwal** - ML/AI & Dataset Management
- **Naman Singhal** - Backend Development
- **Yash Goyal** - Frontend Development & UI/UX

## Tech Stack
- **Frontend**: React.js, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **ML/AI**: Python, scikit-learn, TensorFlow
- **Accessibility**: WCAG 2.1 AA compliance
